# Radio Player DevGeek.IT

Современный веб-радио плеер с анимированным фоном и стеклянным дизайном.

## Особенности

- Анимированный фон на Canvas с частицами
- Стеклянный/неоновый стиль (glassmorphism)
- 100+ радиостанций Radio Record
- Адаптивный дизайн для ПК и мобильных устройств
- Управление с клавиатуры (пробел, стрелки)
- Сохранение последней станции и уровня громкости
- Обработка ошибок потоков

## Использование

1. Откройте файл `radio-player.html` в браузере.
2. Выберите станцию из списка.
3. Нажмите кнопку воспроизведения.
4. Регулируйте громкость с помощью ползунка.

## Управление с клавиатуры

- **Пробел** - воспроизведение/пауза
- **Стрелки влево/вправо** - переключение станций
- **Стрелки вверх/вниз** - регулировка громкости

## Технологии

- HTML5
- CSS3 (анимации, glassmorphism)
- Vanilla JavaScript
- HTML5 Audio API
- Canvas API
